![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)

An interactive Pokedex web app that fetches and displays all 898 Pokemon from the [PokeAPI](https://pokeapi.co/). Browse Pokemon cards with sprites and types, search by name, and click any card to view detailed info. Built as an extracurricular group project for Zenith (EESC-USP).

## Features

- Displays all 898 Pokemon with sprites, names, IDs, and type badges
- Real-time search filtering by name
- Detailed Pokemon info page on card click
- Background music and sound effects
- Color-coded type badges

## Tech Stack

- **HTML / CSS / JavaScript** (vanilla, no frameworks)
- **PokeAPI** REST API for Pokemon data
- **Font Awesome** for icons

## Getting Started

No build step required -- just open `index.html` in a browser:

```bash
open index.html
```

Or serve it locally:

```bash
npx serve .
```
